### Description

A simple portfolio template for developer/designers built with React. 

### [Preview]

![Alt Text](demo.gif)

### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap


### Setup

Get the code

<pre>git clone https://github.com/ajayp1717/My_Portfolio.git</pre>
 
Install required dependencies

<pre>npm install</pre>


Start the server

<pre>npm start</pre>
